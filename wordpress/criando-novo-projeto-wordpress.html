NOME DURAÇÃO

INTRODUÇÃO

Para iniciarmos o desenvolvimento, temos de baixar: 1) Bootstrap 2) JQuery

BAIXANDO O BOOTSTRAP

Ao acessarmos o site oficial do Bootstrap http://getbootstrap.com/getting-started/, temos a disposição 3 links, são eles respectivamente:
  1. Download Bootstrap
  2. Download Source
  3. Download Sass

Download Bootstrap

Este é o link para o desenvolvimento tradicional, ou seja, para desenvolver-mos os nossos sites utilizando HTML e CSS.

Download Source

Neste link temos o código-fonte do framework Bootstrap. Essa é a versão na qual o Bootstrap é construído, ou seja, através dos códigos desse link é que podemos gerar o CSS, JavaScript e o HTML final.

Se você está iniciando agora no desenvolvimento Web, é recomendado o download da primeira opções, até porque, você precisara de conhecimentos avançados sobre CSS, JavaScript (JQuery).

Download Sass

Neste link temos uma versão do framework Bootstrap para uso com a linguagem Sass. Sass é uma linguagem para desenvolvermos CSS de maneira prática. Então, o código Sass é convertido em CSS.

BAIXANDO O BOOTSTRAP

No site do Bootstrap http://getbootstrap.com/getting-started/ clique na primeira opção, de nome "Download Bootstrap". Em seguida, será iniciado o download de um arquivo *.zip cujo nome é bootstrap + a versão do framework.

Terminado o download descompacte e renomeie a pasta raiz que será gerada para "bstrap". Feito isso, você deverá ter uma estrutura de diretórios semelhante a:

-bstrap/
├── css/ ├── js/ └── fonts/

BAIXANDO O JQuery

Para fazer o download do JQuery, acesse o site oficial http://jquery.com/download/ e baixe a última versão da biblioca. Tome cuidado para não baixar a versão 1.x, até porque, essa será a primeira opção de download. Então, certifique-se de fazer o download da versão 2.x.

A diferença entre as versões 1.x e 2.x é o suporte ao navegador Internet Explorer versões 6, 7 e 8. A API entre ambas versões é a praticamente a mesma, assim o que diferencia as duas versões são o suporte aos navegadores antigos e que já estão em desuso. Então, salvo se o seu projeto realmente precisar suportar os navegadores antigos é que você deve utilizar a versão 1.x.

Clique no link "Download the compressed, production jQuery 2.x" e em seguida, será iniciado o download de um arquivo com a extensão *.js.

Observe que para cada versão do JQuery há 3 arquivos disponível para download, são eles: - Download the compressed, production jQuery 1.x/2.x - Download da versão comprimida/compactada jQuery 1.x/2.x

  • Download the uncompressed, development jQuery 1.x/2.x
  • Download da versão descomprimida/descompactada jQuery 1.x/2.x
  • Download the map file for jQuery 1.x/2.x
  • Download do arquivo de mapeamento for jQuery

ESTRUTURA DE PROJETO

Cada projeto em que formos utilizar o framework bootstrap, terá a estrutura de diretórios e os arquivos conforme vimos acima. E também, em cada arquivo HTML que utilizará o framework bootstrap, será necessário fazer referência aos arquivos contidos nessa estrutura.
