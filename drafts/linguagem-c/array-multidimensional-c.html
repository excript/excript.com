ARRAYS MULTIDIMENSIONAIS

Estudamos Arrays unidimensionais, agora, iniciaremos o estudo dos Array que possuem duas ou mais dimensões.

Aprendemos que estruturas unidimensionais são acessada através de um único índice. Agora, quando estivermos trabalhando com Arrays multidimensionais, precisaremos definir o índice corresponde a cada dimensão, isto é, arrays bi-dimensionais terão 2 índices, Array tri-dimensionais, terão 3 índices e assim por diante.

A seguir, temos o esquema para manipulação de estruturas bi-dimensionais:

<tipo> <nome> [linhas] [colunas];

char monitor[800][600];
int tabuleiro_xadrez[7][7];
double tabela[10][5];

Arrays com várias dimensões são Arrays que contém Arrays. Ou seja, imagine um Array unidimensional onde cada elemento deste Array é um outro Array com uma quantidade de elementos. Esse tipo de estrutura será acessada por 2 ou mais índices, onde o primeiro índice é o elemento do primeiro Array e o segundo índice, é o elemento do Array contido na posição informada no primeiro índice.

Por essa razão é que manipular Arrays de várias dimensões é feita de forma igual a como manipulamos Arrays de uma dimensão. A diferença agora é que nós temos que informar um segundo índice, que corresponde a um elemento da nossa lista de Arrays.

INICIALIZANDO ARRAYS MULTIDIMENSIONAIS

int matriz[2][2] = {20, 21, 22, 23, 24, 25, 26, 27, 28};

O Array declarado é um Arrays de 3x3, ou seja, um Array com 9 elementos.

No Array declarado anteriormente, nós definimos uma lista de valores de forma plana. Logo, o compilador ira atribuir os valores para as respectivas posições. Uma outra forma de declaração é utilizando estruturas dentro de estruturas:

int matriz[][] = {  {20, 21, 22},
{23, 24, 25},
{26, 27, 28}};

int matriz[][] = { {20, 21, 22}, {23, 24, 25}, {26, 27, 28} };

Quando trabalhamos com Arrays muito grandes, dificilmente iremos escrever os valores em tempo de desenvolvimento. Uma forma que temos para inicializar um Array de inteiros com o valor igual a zero, é da seguinte maneira:

int matriz[2][2] = { 0.0 };

ARRAYS COM TRÊS OU MAIS DIMENSÕES

Na prática, poucas serão as vezes em que precisaremos de estruturas com mais de 2 dimensões. Somente aplicações bastante especificas é que fazer bastante uso dessas estruturas. O C não estabelece qualquer limite para a quantidade de dimensões, assim, fica a nossa critério a maneira como iremos declarar os Array.

int embalagem[2][2][2];

Uma exemplo de utilização, seria a necessidade de armazenar as dimensões de um pacote que será enviado a um cliente. Assim, temos que um pacote possui Altura x Largura x Profundidade. Essa é uma excelente situação para treinarmos a manipulação de estruturas multidimensionais.

Porém, nessas situações é recomendado a utilização de uma outra estrutura, ou seja, nós trabalharíamos com um Array onde cada elemento é uma estrutura que nós definimos. Essa estrutura contém 3 variáveis para armazenarmos os 3 valores. Obviamente, poderíamos ter uma estrutura com um único Array com 3 elementos.
